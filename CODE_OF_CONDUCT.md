# Code of Conduct

GoodNet adopts the **Contributor Covenant, version 2.1**, verbatim
from <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
The covenant defines the standards of behaviour expected in all
project spaces — issues, pull requests, discussions, mailing list,
chat rooms — and the responsibilities of project maintainers.

## Scope

The covenant applies to:

- This repository (issues, PRs, code review).
- Any official GoodNet communication channel (mailing list, chat,
  public release announcements).
- Public events where a maintainer represents the project.
- Out-of-tree GoodNet plugin repositories that opt into the
  covenant by reference.

## Enforcement contact

Reports of covenant violations go to **<conduct@goodnet.io>**. The
mailbox is read by the core maintainer team; reports are kept
confidential. Reporters receive an acknowledgement within 72 hours
and a follow-up within seven calendar days describing the
investigation outcome and any action taken.

The enforcement timeline mirrors the security disclosure timeline
in `SECURITY.md`. Conduct violations are resolved with the
remedies the Contributor Covenant 2.1 prescribes (warning,
temporary or permanent ban from project spaces) plus the local
remedy of revoking commit access.
