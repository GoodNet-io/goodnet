# Plugin pages

Auto-generated entry per plugin shipped in tree. Each page
summarises the plugin's source-tree README and exposes a
livedoc placeholder that picks up scheme, composer capability,
and source location at every `make livedoc` run.

Plugin source itself lives in `plugins/<kind>/<name>/`, each in
its own standalone git checkout. These docs/ pages are the
kernel-monorepo's window onto that ecosystem.
